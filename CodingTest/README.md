## 알고리즘 성능 분석
### BIG-O 표기법
알고리즘 분석에서 불필요한 함수를 제거하여 간단하게 상한점근으로 표기한다
- O(1)
- O(log n) : 로그 복잡도
- O(n) : 선형 복잡도
- O(n^2) : 2차 복잡도
- O(2^n) : 기하급수적 복잡도

### 시간복잡도
연산의 종류에 관계 없이 한 번의 연산을 1이라고 가정하고 연산횟수를 계산한다

### 공간복잡도
프로그램을 시작하고 완료하는데 필요한 자원 공간의 양
- 총 공간 요구 = 고정 공간 요구 + 가변 공간 요구 [S(P) = c + Sp(n)]
